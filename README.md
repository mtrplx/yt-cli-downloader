# Youtube to mp3 (yottomp3)

### Used technologies
  -  [Node.js](https://nodejs.org/es/)
  -  [Figlet](https://github.com/cmatsuoka/figlet)
  -  [ConfigStore](https://github.com/yeoman/configstore)
  -  [Chalk](https://github.com/chalk/chalk)
  -  [Node ytdl-core](https://github.com/fent/node-ytdl-core)
  -  [Node fluent ffmpeg](https://github.com/fluent-ffmpeg/node-fluent-ffmpeg)
  -  [Readline](https://github.com/maleck13/readline)
  -  [Inquirer](https://github.com/SBoudrias/Inquirer.js/)

### Instalation

Using ```npm install -g yottomp3``` or opening the folder on the console and executing ```npm install -g .```
